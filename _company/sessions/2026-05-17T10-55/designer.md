# 🎬 유튜브 영상 시각 자료 디자인 브리프: 가치관 충돌 극복의 심리학

**시스템 적용:** Deep Navy (주조색/배경), Warm Peach (강조색/액센트)
**브랜드 톤 유지:** 전문적이면서 친근함 (Professional yet friendly)

## 1. 전체 시각 시스템 정의 (Visual System Definition)

| 요소 | 설정 값 | 용도 및 의도 |
| :--- | :--- | :--- |
| **Primary Color (Deep Navy)** | `#0A192F` | 배경, 주요 텍스트 영역. 깊이감과 전문성 부여. |
| **Accent Color (Warm Peach)** | `#FFAD9D` | 핵심 지표 강조, CTA 버튼, 중요한 통찰 포인트 표시. 친근함과 따뜻함 부여. |
| **Secondary Color (Light Gray)** | `#CCD6E1` | 보조 텍스트, 그래프 배경 등 차분한 정보 전달 영역. |
| **Typography (Main Title)** | Pretendard Bold / Black | 강력하고 전문적인 제목 강조. |
| **Typography (Body Text)** | Pretendard Medium / Regular | 친근하면서도 가독성이 높은 본문 설명. |
| **Data Visualization Style** | 레이더 차트 및 비교 막대 그래프 중심 | 심리 지표와 관계의 복잡성을 시각적으로 직관화. |

## 2. 섹션별 시각 자료 상세 브리프 (Scene-by-Scene Visual Brief)

### 📌 Section 1: 🔥 강력한 후크 (Hook) (0:00 - 0:30)

**목표:** 즉각적인 공감 및 몰입 유도.
**시각 컨셉:** 고통과 대비의 극대화.

| 요소 | 디자인 지침 | 세부 적용 사항 |
| :--- | :--- | :--- |
| **배경** | Full Screen Deep Navy (`#0A192F`) | 깊고 차분한 분위기 조성. |
| **핵심 텍스트 (Hook)** | Warm Peach 강조 (`#FFAD9D`) | "당신의 연애가 망가진 진짜 이유?" 문구를 가장 크게 배치. |
| **시각 효과** | 미세한 입체감(Depth) 효과 적용. 고통을 상징하는 추상적인 선이나 그림을 배경에 은은하게 오버레이. |

### 📌 Section 2: 💡 문제 제기: 가치관 충돌의 본질 (0:30 - 1:30)

**목표:** 문제의 원인을 심리적 차원으로 끌어올리기.
**시각 컨셉:** 구조화 및 명료화.

| 요소 | 디자인 지침 | 세부 적용 사항 |
| :--- | :--- | :--- |
| **레이아웃** | 2분할 레이아웃 (좌: 문제 정의, 우: 가치관의 역할) | 화면을 논리적으로 분리하여 분석적인 느낌 부여. |
| **핵심 메시지 박스** | Light Gray 배경 (`#CCD6E1`) 사용. | "관계 실패는 외부가 아닌 내부에서 시작된다"와 같은 핵심 명제를 별도 박스로 강조. |
| **키워드 강조** | '가치관 충돌', '내면의 불일치' 등의 키워드는 Warm Peach로 하이라이트 처리. |

### 📌 Section 3: 🔬 심화 분석: 심리 지표와 사례 통합 (S-A-A) (1:30 - 4:00)

**목표:** 이론(심리 지표)과 실제 사례(S-A-A)를 연결하여 고통의 실체를 보여줌. **가장 중요한 데이터 시각화 구간.**
**시각 컨셉:** 복합적인 관계와 상태를 명확히 표현하는 차트 중심.

| 요소 | 디자인 지침 | 세부 적용 사항 |
| :--- | :--- | :--- |
| **차트 유형** | **레이더 차트 (Radar Chart)** 및 **비교 막대 그래프 (Bar Chart)** 혼합 사용. | M1~M4 심리 지표의 각 축(자존감, 안정감, 욕구 충족 등)을 레이더 차트로 표현하여 복잡한 관계를 한눈에 파악하게 함. |
| **사례 통합** | S-A-A 사례는 별도의 카드 또는 박스 형태로 배치. | 각 사례는 해당 심리 지표(M3/M4)와 직접적으로 연결되도록 시각적 흐름을 설계. |
| **색상 사용** | 레이더 차트의 축은 Light Gray, 실제 점(데이터 포인트)은 Warm Peach로 채워 시선 집중 유도. |

### 📌 Section 4: 🛠️ 해결 전략: 행동 변화 로드맵 (4:00 - 6:30)

**목표:** 추상적인 분석에서 구체적이고 실행 가능한 행동으로 전환.
**시각 컨셉:** 단계별 진행(Step-by-Step) 및 명확한 로드맵 제시.

| 요소 | 디자인 지침 | 세부 적용 사항 |
| :--- | :--- | :--- |
| **레이아웃** | 수평 또는 수직의 **3단계 프로세스 흐름도 (Flowchart)** | '인지 $\rightarrow$ 분석 $\rightarrow$ 행동'의 명확한 순서를 강조. |
| **각 단계 디자인** | 각 단계(Step)는 네모난 카드 형태로 구성. | 1단계: 인지, 2단계: 객관화, 3단계: 실행을 명시하고, 각 단계 옆에 Warm Peach 색상의 아이콘 배치. |
| **액션 강조** | '행동 전략' 부분은 가장 밝은 배경(Light Gray)과 대비되는 Deep Navy 텍스트를 사용하여 강력한 실행 의지를 부여. |

### 📌 Section 5: ✅ 최종 정리 및 CTA (6:30 - 끝)

**목표:** 메시지 재강조 및 행동 촉구.
**시각 컨셉:** 자신감 있는 마무리와 명확한 다음 단계 제시.

| 요소 | 디자인 지침 | 세부 적용 사항 |
| :--- | :--- | :--- |
| **요약 메시지** | 핵심 3가지 포인트를 Bullet Point로 간결하게 재배치. | 가장 중요한 행동 전략만 남기고 시각적 복잡성을 제거. |
| **CTA 버튼** | Warm Peach 배경에 Deep Navy 텍스트의 대비를 극대화하여 배치. | "지금 바로 시작하세요"와 같은 CTA는 눈에 띄게, 클릭을 유도하도록 디자인. |
