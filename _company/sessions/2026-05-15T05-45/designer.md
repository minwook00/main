## 🎬 영상 제작 최종 비주얼 가이드라인

### 1. 브랜드 스타일 확정 (Brand Style Finalization)

| 요소 | 세부 정의 | 적용 규칙 |
| :--- | :--- | :--- |
| **메인 컬러** | Deep Navy (전문성, 신뢰감) | `#0A1931` (주요 배경, 텍스트 강조) |
| **보조 컬러** | Warm Peach (친근함, 따뜻함) | `#FFB88C` (강조 박스, CTA 버튼, 하이라이트) |
| **포인트 컬러** | Light Gray/Off-White | `#F5F5F5` (배경 및 보조 텍스트 영역) |
| **타이포그래피** | 제목 (Bold & Strong) | Noto Sans Bold 또는 Pretendard Black |
| | 본문 (Readability) | Noto Sans Regular |

### 2. 썸네일 디자인 컨셉 최종 확정 및 가이드

세 가지 컨셉 중, **컨셉 A**를 메인으로 채택하며, 이를 기반으로 모든 썸네일 제작 시 다음 규칙을 준수해야 합니다.

| 구분 | 컨셉 A (추천) | 적용 디테일 |
| :--- | :--- | :--- |
| **레이아웃** | 중앙 집중형 대비 강조 (Visual Contrast Focus) | 화면의 60% 이상을 Deep Navy 배경으로 설정하고, Warm Peach로 핵심 메시지(가치관 충돌 지표 M1~M4)를 시각화. |
| **핵심 요소** | 심리적 갈등 표현 | '불일치'나 '충돌'을 상징하는 대비되는 색상 블록이나 아이콘 사용. |
| **타이포그래피** | 메시지 선명도 극대화 | 핵심 질문(후킹 문구)은 흰색 또는 연한 Peach으로 강조하고, 지표 설명은 Deep Navy 텍스트로 명확하게 구분. |
| **감성** | 전문적이지만 공감을 유도하는 따뜻함 | 차가운 Navy와 따뜻한 Peach의 대비를 통해 '분석'과 '공감'을 동시에 전달. |

### 3. 영상 내 인포그래픽 (차트/그래프) 디자인 가이드라인

핵심 콘텐츠인 **'가치관 불일치 차트 (M1~M4)'**는 시청자가 즉각적으로 이해하고 몰입할 수 있도록 디자인되어야 합니다.

*   **레이아웃:** 수평 또는 세로형으로 명확하게 구분하여 각 지표(M1, M2, M3, M4)를 독립적인 단위로 강조합니다.
*   **색상 적용:**
    *   **기본 배경:** Deep Navy (`#0A1931`)
    *   **차트 선/구분선:** Warm Peach (`#FFB88C`) 또는 밝은 회색.
    *   **데이터 영역:** 지표별로 미묘하게 다른 톤의 Peach 색상을 사용하여, 상대방과의 '불일치 정도'를 시각적으로 직관화합니다. (예: M1이 가장 큰 불일치를 나타내면 가장 채도가 높은 Peach 사용)
*   **폰트:** 데이터 수치는 크고 명확하게 표시하며, 설명 텍스트는 본문과 동일한 톤의 흰색 또는 밝은 회색을 사용하여 가독성을 최우선으로 합니다.

### 4. 최종 비주얼 점검 리스트 (Production Checklist)

영상 편집 단계에서 다음 사항들이 반드시 확인되어야 합니다.

1.  **컬러 일관성:** 모든 인포그래픽과 자막에 Deep Navy, Warm Peach의 톤이 일관되게 적용되었는가?
2.  **후킹 시각화:** 영상 도입부(후킹 구간)에서 '가치관'이라는 단어가 화면에 가장 강력하게 각인되는지 확인했는가? (썸네일 컨셉과 연동)
3.  **차트 가독성:** M1~M4 지표 차트는 5초 이내에 핵심 메시지를 전달할 수 있도록 시각적 계층 구조가 명확한가?
4.  **톤앤매너:** 전체 영상이 전문적이면서도 남성 시청자가 편안하게 접근할 수 있는 친근함을 유지하는가? (차가움과 따뜻함의 균형)
